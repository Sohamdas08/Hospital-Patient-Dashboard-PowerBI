# 🏥 Hospital-Patient-Dashboard-PowerBI


## 📌 Overview
This project presents an interactive Power BI dashboard for analyzing hospital operations, including patient details, billing, and medicine usage.

---

## 🎯 Objectives
- Analyze patient admission trends  
- Track hospital revenue  
- Monitor medicine usage  
- Evaluate patient satisfaction  

---

## 📁 Dataset
Dataset used in this project is available in the `/Data` folder.

---

## 🧮 DAX Calculations
All measures and calculated tables are available in:
👉 `/DAX/DAX_Measures.md`
